# Kickoff Atlas

Kickoff Atlas is a frontend web application that helps users discover upcoming football matches in a selected city and filter them by date or team.

## Tech Stack
- React
- Vite
- Tailwind CSS
- React Router

## Features
- Browse football matches by city
- Filter matches by date
- Filter matches by team
- View match details
- Redirect to external ticket provider

## Accessibility
- Keyboard navigation supported
- Focus visibility verified
- Semantic HTML structure used
- Color contrast considered

## Deployment
The application is deployed using Vercel.
